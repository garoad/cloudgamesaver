# CloudGameSaver (v0.1.4)

드롭박스(Dropbox)를 활용한 게임 세이브 데이터 자동 동기화 툴입니다. 
Tauri(v2)와 React(TypeScript)를 기반으로 제작되었습니다.

## 주요 기능 (v0.1.4 업데이트 사항)

*   **지능형 양방향 동기화**: 로컬 파일과 드롭박스 서버 파일의 해시 및 수정 시간을 비교하여 가장 최신 파일을 자동으로 선택해 동기화합니다.
*   **영구 세션 관리 (Refresh Token)**: 4시간마다 만료되던 기존 방식에서 탈피하여, 리프레시 토큰을 통해 세션을 자동으로 갱신합니다. (최초 1회 재로그인 필요)
*   **Dropbox 표준 해시 검증**: 4MB 블록 단위 SHA256 스트리밍 해시 계산을 통해 파일 내용이 실제로 다른지 완벽하게 검증합니다. (빈 파일 처리 포함)
*   **상세 동기화 피드백**: 동기화 진행 상황을 실시간 진행바(ProgressBar)와 현재 처리 중인 파일명을 통해 확인할 수 있습니다.
*   **안정적인 네트워크 처리**: 동기화 중 토큰 만료(401 Unauthorized)가 발생해도 중단 없이 즉시 갱신하여 작업을 재개합니다.

## 사용 방법

1.  **드롭박스 연결**: 우측 상단의 '드롭박스 연결하기' 버튼을 눌러 인증을 완료합니다.
2.  **게임 추가**: 게임 이름과 PC의 세이브 폴더를 지정하여 목록에 등록합니다.
3.  **동기화**: '동기화 시작' 버튼을 누르면 모든 활성화된 게임의 세이브 데이터가 클라우드와 정합성을 맞춥니다.

## 기술 스택

*   **Backend**: Rust (Tauri v2, tokio, reqwest, chrono, sha2)
*   **Frontend**: React v19, TypeScript, Vite
*   **Cloud API**: Dropbox OAuth2 (PKCE + Refresh Token)

## 주의 사항
*   최초 1회 재인증을 진행해야 리프레시 토큰이 정상적으로 발급되어 세션이 유지됩니다.
*   로컬 폴더를 잘못 지정할 경우 기존 파일이 덮어씌워질 수 있으니 주의하시기 바랍니다.
